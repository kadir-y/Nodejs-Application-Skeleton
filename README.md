
# Nodejs Application Skeleton

Backend service created using express framework.




## Installation

Install my-project with npm

```bash
  cd Nodejs-Application-Skeleton
```
```bash
  npm install Nodejs-Application-Skeleton
```
    
## Deployment

To deploy this project run

```bash
  cd Nodejs-Application-Skeleton
```
```bash
  npm run start
```
